# Cyberpunk Mega-Corporation Logo Match

A JavaScript memory matching application based on the dystopian future.  Users must get above at least 50% accuracy in order to truly win the game.

# Technologies Used
- JavaScript
- jQuery
- HTML5
- CSS3
